# static-site-generator

## How to run the web server?

```
python server.py --dir public
```

## How to run the static site generator?

```
sh main.sh
```

or

```
./main.sh
```

## How to run Unit Tests ?

```
sh test.sh
```

or

```
./test.sh
```

### Permission denied?

You need to make it executable:

```
chmod +x ./main.sh
chmod +x ./test.sh
```
